Diferencias de Significado entre Etiquetas HTML

En HTML, hay etiquetas que se ven igual en la página, pero que para el navegador y para los buscadores como Google, significan cosas diferentes. Entender esto es importante para que la página sea accesible y se posicione mejor.

<b> vs. <strong>

  • Cómo se ven: Ambas etiquetas ponen el texto en negrita.
  • Qué significan: La etiqueta <b> solo resalta el texto visualmente para llamar la atención, pero no le da más importancia. En cambio, <strong> le dice al navegador que ese texto es muy importante.
  • Para los buscadores: Google le presta más atención al texto dentro de <strong> porque entiende que es una parte fundamental del contenido.

<i> vs. <em>

  • Cómo se ven: Las dos etiquetas muestran el texto en cursiva.
  • Qué significan: La etiqueta <i> se usa para diferenciar un texto del resto, por ejemplo, si es una palabra en otro idioma. La etiqueta <em> se usa para dar un énfasis que cambia el significado de lo que se está diciendo.
  • Para los buscadores: Usar <em> ayuda a que los buscadores entiendan mejor cuál es la intención o el punto clave de una frase.

<s> vs. <del>

  • Cómo se ven: Ambas etiquetas sirven para tachar el texto.
  • Qué significan: La etiqueta <s> se usa para marcar algo que ya no es correcto o que no aplica, como un precio viejo. Por otro lado, <del> se usa para mostrar que un texto fue eliminado, como en una corrección.
  • Para los buscadores: La etiqueta <del> es más clara para los buscadores, pues les indica que hubo un cambio o una edición en el contenido de la página.

Elementos de Línea y Elementos de Bloque

Elementos de Bloque

Un elemento de bloque siempre empieza en una línea nueva y ocupa todo el ancho que tiene disponible en la página. Sirven para crear la estructura principal del contenido.

  • <div>: Un contenedor o "caja" para agrupar otros elementos.
  • <p>: Sirve para escribir un párrafo de texto.
  • <h1>, <h2>, etc.: Son los títulos y subtítulos de la página.
  • <ul>: Crea una lista con viñetas.
  • <form>: Define un formulario para que los usuarios introduzcan datos.

Elementos de Línea

Un elemento de línea no crea un salto de línea y solo ocupa el espacio justo de su contenido. Normalmente se usan dentro de un elemento de bloque para marcar algo específico.

  • <span>: Se usa para agrupar o aplicar estilos a una pequeña parte de un texto.
  • <a>: Crea un enlace o hipervínculo a otra página.
  • <strong>: Marca un texto como importante.
  • <img>: Inserta una imagen en la página.
  • <input>: Es un campo de un formulario para que el usuario escriba algo.
